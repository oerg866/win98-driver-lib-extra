VIA Vinyl AC'97 WDM Driver (v700b)

Version: 6.14.01.4200
Release Date: 2007-06-27
Target OS: Windows 98 Second Edition
Driver Model: WDM (Windows Driver Model)

Description
-----------
This package contains the VIA Vinyl AC'97 WDM audio driver files
extracted from the official VIA v700b distribution package.

Only Windows 9x-compatible driver files are included.
No installer, MSI, setup executable or control panel software
is provided in this package.

Tested Hardware
---------------
Chipset:
- VIA CN700 Northbridge
- VIA VT8237 Southbridge (AC'97 Audio Controller)

Hardware ID:
PCI\VEN_1106&DEV_3059

Installation
------------
1. Open Device Manager.
2. Locate "Multimedia Audio Controller".
3. Choose "Update Driver".
4. Select "Specify location".
5. Point to this folder.

Notes
-----
- This is a WDM driver for Windows 98 SE.
- The original VIA control panel (ADeck) is NOT included.
- Extracted from official VIA Vinyl v700b package.
- Intended for manual driver installation.
